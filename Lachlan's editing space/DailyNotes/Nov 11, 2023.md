Collatz Conjecture
Every positive integer applied to the rule (if odd, multiply by 3 and add 1, if even divide by 2) will eventually end up in the 4-2-1 loop. 


```python
n = 3
while n > 1:
	if n % 2 == 0:
		n = n / 2
	else:
		n = 3*n + 1
```
